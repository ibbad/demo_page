Securebox - Safeguard Network Edge with Lightweight Virtual Middleboxes

Video


Highlights

We demonstrate a novel system for securing smart home and small enterprise networks and IoT environments by using cloud-based traffic analysis services and virtualized middleboxes. The key features of this proposal include:

  • "Everything in a box" security solution
  • Unifying SDN and Cloud-based intelligence
  • Easy to deploy and operate and Transparent to users
  • Openness and Programmability through pluggable virtualized modules
  • Enabling collaborative analytics for mitigating security threats
